# Iris Flower Classification

## 📌 Objective
This project aims to classify Iris flowers into three species (Setosa, Versicolor, and Virginica) using Machine Learning algorithms.

## 📂 Dataset
The Iris dataset contains measurements of iris flowers:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species
